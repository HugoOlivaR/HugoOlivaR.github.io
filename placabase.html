PLACAS BASE


En este artículo de placas base vamos a hablar un poco de dos placas base en específico, con el socket AM4, es decir, una para procesadores AMD y otra placa con LGA 1151 para procesadores Intel.
Pero antes de nada un pequeño repaso a las placas base.
¿Que es una placa base? Una placa base o placa madre es uno de los componentes principales de un ordenador, es una PCB con un circuito impreso la cual tiene diversos componentes para hacer funcionar tu ordenador. En la placa base se conectan todos los demás componentes como el microprocesador, la tarjeta gráfica o los discos duros para que todo funcione todo conectado entre sí.




MSI B450 Gaming Plus MAX



La primera placa base de la que vamos a hablar es la MSI B450 Gaming Plus MAX. Es una placa para procesadores con AM4 es decir procesadores de AMD.
Esta placa base, ahora mismo es una de las mejores relación calidad precio del mercado, es de clase ATX. Como ya hemos nombrado es una placa con el socket AM4 y un chipset B450 de AMD, de ahí su nombre. Esta placa tiene conectores USB 3.2 GEN2 para la mayor velocidad de tus periféricos y 6 puertos USB 2.0 extras.
Tiene 4 ranuras DIMM DDR4 con una frecuencia de hasta 4133 Mhz OC, una muy alta frecuencia perfecta para sacar mas FPS en videojuegos.
Tiene PCI-E 3.0 que es compatible con tarjetas gráficas como las de la serie 2000 de Nvidia perfectas para gaming y trabajos de modelado 3D.
Esta placa base es una placa perfecta para gaming para el precio que tiene, este ronda los 97€, teniendo en cuenta que está preparada para procesadores de gama media y gama alta, es un precio muy asequible.




Asus Prime B365M-A



La segunda placa de la que vamos ha hablar es la Asus Prime B365M-A.
Estas placas de ASUS Prime 300 Series, proporcionan la base sólida necesaria para su primera construcción, además de flexibilidad para crecer con sus ambiciones.
Esta placa se caracteriza por tener un socket de procesador LGA 1151 (Zócalo H4).
Además acepta hasta procesadores de novena generación y la Bios es muy sencilla y muy completa.
Tiene como tipos de memoria compatibles: DDR4-SDRAM, y un factor de forma micro ATX, también incorpora puerto USB tipo-c y varias entradas sata para ssd o hdd.
Y por último, esta placa base ahora mismo se encuentra en el mercado por 77,98€, viendo que es una placa homework, networking, station y gaming, su precio es bastante bueno.
